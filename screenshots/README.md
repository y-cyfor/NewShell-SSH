# 截图目录

此目录用于存放 NewShell 项目的截图，这些截图会在主 README.md 中引用。

## 需要的截图

请提供以下截图文件：

1. **main.png** - 主界面完整截图
2. **terminal.png** - 终端界面截图
3. **filemanager.png** - 文件管理器截图

## 截图要求

- 格式：PNG
- 分辨率：建议 1920x1080 或更高
- 内容：展示 NewShell 的主要功能
- 主题：建议使用暗色主题（默认）

## 如何获取截图

1. 运行 NewShell 应用
2. 连接服务器
3. 调整窗口大小
4. 使用系统截图工具（如 Windows 的 Snipping Tool）
5. 保存到此目录

## 文件命名

- 主界面：`main.png`
- 终端：`terminal.png`
- 文件管理器：`filemanager.png`

## 注意事项

- 截图应清晰展示界面元素
- 避免包含敏感信息（如密码、密钥等）
- 确保截图质量足够高，能够清晰显示文字
